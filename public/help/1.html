This guide will help you understand what Ollama models, Hugging Face Spaces, and AI websites are - and how to use them inside this app.

ü§ñ What Are Ollama Models?

Ollama lets you run powerful AI models locally on your computer. These models can chat, summarize, write code, and more - all without needing an internet connection.

  • Pull: Download a model to your system
  • Delete: Remove a model you no longer need
  • Chat: Open a chat window with the selected model

Models are stored locally and run directly on your machine, giving you privacy and speed.

  • Save: Bookmark your favorite Spaces

  • Delete: Remove saved Spaces you no longer use

Spaces are hosted online, but this app embeds them so you can use them without switching tabs.

üåç What About Websites?

Some tools live on regular websites. This app lets you save and launch them in a separate Chromium window for a focused experience.

  • Save: Bookmark useful AI websites
  • Launch: Open them in a new window
  • Delete: Remove saved sites
  • First time? Try pulling a model and launching a Space to get started!

‚ùì Frequently Asked Questions

Q: Do I need an internet connection?
A: Only for downloading models or accessing online Spaces/websites. Once pulled, Ollama models run locally.

Q: Why does a website open in a new window?
A: Some websites block embedding for security reasons. The app opens them in a separate Chromium window to bypass that.

Q: Can I use multiple models at once?
A: Yes! You can chat with different models in separate windows.
