The app stores chat sessions locally so you can continue conversations later. Sessions include the model, history and a name.

Saving and loading

  • Use the session menu to save the current conversation.
  • Favorites show at the top of your session list for quick access.
  • Sessions are stored in the application's data directory.

Export / Import

  • You can export session files via the app menu (if enabled).
  • Imported sessions are treated as local data and do not automatically sync.
